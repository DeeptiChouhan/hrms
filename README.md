# HRMS Automation Framework (Playwright + Pytest)

## 🔹 Overview
This project is a scalable test automation framework built using Playwright (Python) and Pytest for HRMS application testing.

## 🔹 Features
- Page Object Model (POM)
- Base Page abstraction
- Data-driven testing (JSON)
- Pytest fixtures
- HTML reporting
- CI/CD ready (GitHub Actions)

## 🔹 Tech Stack
- Python
- Playwright
- Pytest

## 🔹 Test Coverage
- Login functionality
- Negative scenarios (invalid credentials)
- Validation checks (empty fields)

## 🔹 How to Run
```bash
pip install -r requirements.txt
pytest
